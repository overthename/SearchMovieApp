# ShoppingApp
### 기술스택
* MVVM 패턴
* retrofit2
* Moshi
* Hilt
* Coroutine
* Room
* Navigation
<br>

### 기능
* 네이버 쇼핑 검색 API를 사용해 검색어에 따른 쇼핑 목록을 제공
* 쇼핑목록은 무한스크롤로 다음 목록을 불러올 수 있음
* 좋아요 기능과 좋아요 화면 제공
